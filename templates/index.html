Demo web app

This is a demo of my political bias classier program. It allows you run the program right here in the browser. Simply type in a sequence of text in the box below, hit submit, and wait until the output is shown. The output tells if the input text is partisan or neutral along with a percentage value. Be warned though, since the machine learning model is quite large (> 200GB), it does some time for the model process the input. Enjoy!
